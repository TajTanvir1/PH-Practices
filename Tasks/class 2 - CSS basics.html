Class 2 - Practices

Practice 2.1 (CSS Types Challenge)

  • Create an Html document with three paragraph.
  • Apply CSS style to each paragraph using inline, internal and external Styles.

Practice 2.2 (Text Stylling Challenge)

  • Desing A webpage with a heading, paragraph and a list.
  • Apply CSS styles to change the text color, alignment, font size, font-family and font style for different elements.

Practice 2.3 (Selector Challenge)

  • Create an HTML document with various elements including headings, paragraph and divs.
  • Use different CSS selectors (id, class, universal tag) to apply unique styles to specific elements.

Practice 2.4 (Background Styling Challenge)

  • Design a webpage with multiple sections/divs.
  • Apply CSS styles to set different Background colors, imagers, repeats, positions and sizes for each section.

Practice 2.5 (Borders Challenge)

  • Create a webpage with various elements such as divs, paragraph and imagers
  • Apply CSS styles to add borders with different widths, color, border-radius, and box-shadow effects to the elements.

Practice 2.6 (Display Challenge)

  • Build a webpage with different elements (e.g images, headings, paragraph, button.
  • Apply CSS styles to set the display property to none, hidden, inline, block and inline-block for specific elements.
